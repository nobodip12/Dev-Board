Question-1: What are the different ways to select an element in the DOM?

The ways to select elements in the dom are getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The differences between innerHTML and innerText are that innerHtml gets the full HTML code and returns everything including HTML tags .while innerText only gets text and does not return any HTML tags.

Question-3: What is event delegation in the DOM?

Event Delegation is a simple technique where an event listener is placed on a parent element to capture events that occur on its child elements. Event Delegation is especially used to keep the code short and clean.

Question-4: What is event bubbling in the DOM?

When you click on a child element, the event that is generated is not limited to the child element but also bubbles up on the parent, grandparent, etc.

Question-5: How do you create, add, and remove elements using JavaScript?

Using javascript, add the DOM through createElement and push the code to the id using appendChild. You can do this by using the remove tag with the id you want to remove.
